Abstract:
% abstrakt v rozsahu 80-200 slov v angliètinì; nejedná se však o pøeklad
% zadání bakaláøské práce
The main target of the Origamist application is to aid with creating origami diagram manuals. Recently, the most common methods for creating those manuals are drawing the steps in an image editor, or taking photographs of the folded steps and composing them together (again in an image editor). What Origamist brings is a new alternative to these methods. The manual's author gets the possibility to transfer the sequence of paper folds the manual consists of to the Origamist application, which is able to export several types of output - the most favourite (among origami folders) PDF manuals, but also image manuals (as PNG or SVG), and even as an animation of the folding process. There is also the added value to translate the descriptions of the steps to several languages.

\chapwithtoc{Contents of the attached CD}
Here is the structure of the files located on the attached CD:
\begin{itemize}
\item \emph{.git} The complete Git repository of this project (may be a hidden file)
\item \emph{bin} The compiled .class files.
\item \emph{diagrams} Sample diagrams and export outputs of Origamist.
\item \emph{doc} Documentation.
	\begin{itemize}
	\item \emph{thesis} Sources of this bachelor thesis.
	\item \emph{odd\_things\_and\_how\_to\_fix\_them.txt} Some interesting solutions to various Java-related problems.
	\item \emph{podrobna\_specifikace.pdf} The detailed specification of this project, in Czech.
	\item \emph{roadmap.odt} The roadmap of the project, in Czech.
	\item \emph{specifikace.pdf} The rough specification of this project, in Czech.
	\end{itemize}
\item \emph{javadoc} JavaDoc per-class documentation.
\item \emph{lib} Runtime libraries.
\item \emph{lib-compile-time} Compile-time libraries.
\item \emph{src} The source codes of Origamist.
\item \emph{tests} Resource files for JUnit tests.
\item \emph{.htaccess} A sample Apache server configuration file that enables serving JNLP files with the correct MIME type.
\item \emph{browser.bat.example, browser.sh.example} Templates of shell scripts that run a web browser.
\item \emph{build.xml} The build file (is run by Apache Ant).
\item \emph{java.policy.applet} A needed security policy file.
\item \emph{origami\_editor.jnlp} JNLP file designed for distribution.
\item \emph{origami\_editor.test.jnlp} JNLP files designed for local testing.
\item \emph{origami\_viewer.jnlp} JNLP file designed for distribution.
\item \emph{origami\_viewer.test.jnlp} JNLP files designed for local testing.
\item \emph{Origamist.jar} The main Origamist archive.
\item \emph{OrigamistEditor.jar} The Origamist Editor additional archive.
\item \emph{OrigamistViewer.jar} The Origamist Viewer additional archive.
\item \emph{preview.html} An HTML page running Origamist Viewer as a webpage applet.
\item \emph{preview-editor.html} An HTML page running Origamist Editor as a webpage applet.
\item \emph{README} Instructions on how to run Origamist applications from this CD.
\item \emph{signapplet.bat.example, signapplet.sh.example} Templates of shell scripts that digitally sign JAR files.
\end{itemize}
